## azrdream_music


## 介绍

【vue全家桶练习项目】仿网易云音乐，目前实现了页面基本样式，播放，歌词，歌单，专辑，歌手，搜索，收藏等基础功能。


## 线上demo体验(移动端)

http://www.azrdream.com/music


#### 注意

##### 1、因为服务器没有进行相关配置，为了不影响功能，所以线上demo采用的hash模式

##### 2、代码中实现了预渲染解决SPA应用SEO问题，所以路由采用history模式，部署在服务器后需要解决刷新404问题


## 运行步骤

#### 1、npm install

#### 2、npm run serve
